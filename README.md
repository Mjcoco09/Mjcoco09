## Featured Project

### iMall Marketplace Platform

iMall is a full-stack multi-app marketplace platform built with a Flutter customer app, React admin/business portals, delivery dispatch tools, a driver PWA, and a shared Firebase backend.

Key features include:

- Store and product browsing
- Cart and checkout
- Order history and order status tracking
- Address management with Google Maps/geolocation
- Referrals and reward redemption
- Push notifications and deep links
- Admin and business owner portals
- Store, product, inventory, and order management
- Delivery dispatch, driver assignment, COD reconciliation, and proof-of-delivery workflows
- Firebase Auth, Firestore, Cloud Functions, Storage, Hosting, and Cloud Messaging
- Role-based access control with custom claims
- Firestore security rules, indexes, and shared data models
- Google Gemini AI workflows for product/image tools
- Multilingual support for English, Kurdish, and Arabic
